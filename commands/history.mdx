`designlang history` reads the local snapshot log for a given URL and prints a timeline of every extraction ever run against that site. It is a quick way to answer "when did their primary color change?" or "has the accessibility score improved since we last checked?"

## Usage

```bash
designlang history <url>
```

There are no options. The command reads from the local history store and prints to stdout.

## How snapshots are saved

Every time you run `designlang <url>`, `designlang sync`, or `designlang watch`, a compact snapshot is automatically appended to `~/.designlang/<hostname>.json`. The snapshot records:

- Extraction timestamp
- Primary, secondary, and accent colors
- Font families and type scale size
- Spacing base and scale count
- Shadow and border radius counts
- Component patterns detected
- Accessibility score
- CSS variable count

Snapshots are intentionally compact â€” they store key metrics only, not the full extracted design data. This keeps the history file small even after hundreds of runs.

## Skipping history

Pass `--no-history` to the main `extract` command to run an extraction without saving a snapshot:

```bash
designlang https://stripe.com --no-history
```

This is useful in CI pipelines where you want the output files but don't want to accumulate snapshots.

## Where snapshots are stored

History is stored per-hostname in your home directory:

```text
~/.designlang/
  stripe.com.json
  vercel.com.json
  github.com.json
```

Each file is a JSON array of snapshot objects, sorted oldest-first. You can inspect or delete individual files directly if you need to reset history for a site.

## What the history output shows

`designlang history` prints a markdown-formatted timeline to stdout. For each snapshot it shows the date, color count, primary color, font families, spacing scale size, accessibility score, and CSS variable count. If two or more snapshots exist, it also prints a "Changes Over Time" summary highlighting anything that shifted between the first and most recent snapshot:

```bash
designlang history https://stripe.com
```
